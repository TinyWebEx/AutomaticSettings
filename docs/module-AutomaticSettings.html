(static) setDefaultOptionProvider(defaultOptionCallback) → {void}

Source:
  • internal/OptionsModel.js, line 170

Sets the callback for getting the default options.

See defaultOptionGetterCallback for how the callback needs to behave. You need to call this function before the main init function of AutomaticSettings. However, if you do not want to specify defaults in JS, but just in HTML, you can pass "null" to this and it will not try to request defaults. Pass "undefined" to it to unset it.
